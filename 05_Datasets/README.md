# 标定板图像与测试数据

> 本目录使用 Git LFS 管理，克隆后请执行 `git lfs pull`

## 目录结构
```
05_Datasets/
├── raw_calibration_imgs/     # 左右相机标定采集原图
│   ├── left/                 # 左相机棋盘格/圆点格图像
│   └── right/                # 右相机棋盘格/圆点格图像
└── test_scenarios/           # 边缘极限测试用例
```

## 数据说明
- 标定图像：棋盘格（格数 12×9, 方格边长 ~5mm）或圆点阵列
- 人眼图像：真实设备采集，含瞳孔标注
